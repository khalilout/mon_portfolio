# mon_portfolio

Portfolio statique (HTML/CSS/JS) généré à partir du CV.

## Démarrer en local

Dans ce dossier :

```powershell
python -m http.server 5173
```

Puis ouvre `http://localhost:5173`.

## Mettre à jour le CV

- Remplace `assets/CV_Khalilou_Tandia.pdf` par ton PDF.
- Ajoute tes certifications PDF dans `assets/certs/` (ex : `assets/certs/AWS_Academy_Graduate_Cloud_Foundations.pdf`).
